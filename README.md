Web alternative to [LRCGET](https://github.com/tranxuanthang/lrcget), inspired by [https://github.com/magic-akari/lrc-maker](https://github.com/magic-akari/lrc-maker)

## Prerequisites

- [Bun](https://bun.com/)

## Installation

Install dependencies

```sh
bun install
```

# Formatting

```sh
bun run format
```

# Development

```sh
bun run dev
```

# Building

```sh
bun run build
```
